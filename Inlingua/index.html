Launch Methods and Options

How to Launch the eReader?

  • To launch the eReader, click on the Launch button in the eReader card. You can specify a particular page number to start from.
  • If no page number is specified, the eReader will default to the last page you visited in your previous session.
  • Additionally, you have the option to launch the eReader in an iframe. This will give you an idea of how the eReader looks and feels when embedded in another application.

Embedding eReader in your LMS

  • The eReader supports embedding itself into other applications. This can be achieved with the help of an iframe.
  • Select the option “Launch in Iframe” before clicking the Launch button. This will give you an idea of how the eReader looks and feels when embedded in another application.
  • Technical details are provided below.

Key Highlights

Hotlink Media & Activities

comproDLS eReader supports the following types of hotlinks:

  • External Activities: Link your interactive activities inside the eReader allowing students to perform the activities in context of the book material.
  • Media: Play video and audio within the eReader, with support for transcripts.
  • Jump To Page: Create links that navigate directly to specific pages within the eReader.

Keyboard Support

  • The eReader is designed with keyboard accessibility in mind.
  • The keyboard help menu can be launched using the ? key.
  • To move around the eReader application, use the TAB key

Technical Specifications

Launch Parameters

  • The eReader url comprises of 4 parts baseURL/{Program-Code}/{Product-Class}/{Product-Code}
    • Base URL i.e. https://inlreader.comprodls.com
    • Program Code - Unique code of the program
    • Product Class - Whether the eReader is for Student Edition or Teacher Edition
    • Product Code - Unique code for the title
  • Use the query parameter page in the eReader URL to specify the page number to be launched. For example, to launch page 5 the url will be:
    • https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext?page=5

How to embed the eReader in your application?

  • Create an iframe: Set the src attribute of the iframe to https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext
  • Allow Fullscreen: Set the allowfullscreen attribute in the iframe to enable the eReader to go fullscreen when the user clicks on the fullscreen button.
  • Embed Mode: Set the query parameter embed when launching the eReader in an Iframe.
  • Specify Page Number: Set the query parameter page in the eReader URL to specify the page number to be launched.
  • Sample Iframe: <iframe src="https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext?page=36&embed" allowfullscreen>
