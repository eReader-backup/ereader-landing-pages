Launching the eReader

Standard Launch

  • Click "Launch" in the eReader card to open the eReader.
  • You can start from a specific page by entering the page number.
  • If no page number is provided, the eReader will open to the last page you visited.

Embedding the eReader in Your LMS

  • Embed the eReader into your LMS using an iframe for a seamless integration.
  • Select the “Launch in Iframe” option and click "Launch" to experience the eReader within another application.

For detailed technical information, see the documentation below.

Key Highlights

Hotlinking Media & Activities

comproDLSTM eReader supports the following types of hotlinks:

  • Activities: Embed interactive activities directly within the eReader, enabling students to engage with activities in the context of the book material.
  • Media Integration: Seamlessly play videos and audio within the eReader, complete with transcript support for enhanced accessibility.
  • Jump to Page: Create direct links to specific pages, allowing quick navigation within the eReader.

Keyboard Support

  • The eReader is designed for optimal keyboard accessibility.
  • Launch the keyboard help menu by pressing the ? key.
  • Easily navigate through the eReader application using the TAB key.

Technical Specifications

Launch Parameters

  • The eReader url comprises of 4 parts baseURL/{Program-Code}/{Product-Class}/{Product-Code}
    • Base URL i.e. https://inlreader.comprodls.com
    • Program Code - Unique code of the program
    • Product Class - Specifies if the eReader is for Student Edition or Teacher Edition
    • Product Code - Unique code for the title
  • To specify a page number to be launched, use the "page" query parameter in the eReader URL
  • For example, to launch the page 5:
    • https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext?page=5

Embedding the eReader in Your Application

  • Create an iframe: Set the src attribute of the iframe to https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext
  • Allow Fullscreen: Set the allowfullscreen attribute in the iframe to enable the eReader to go fullscreen when the user clicks on the fullscreen button.
  • Embed Mode: Set the query parameter embed when launching the eReader in an Iframe.
  • Specify Page Number: Set the query parameter page in the eReader URL to specify the page number to be launched.
  • Sample Iframe: <iframe src="https://inlreader.comprodls.com/demo/student-edition/interkulturel_1_vtext?page=36&embed" allowfullscreen></iframe>
