Hotlinking Media & Activities

comproDLSTM eReader supports the following types of hotlinks:

  • Activities: Embed interactive activities directly within the eReader, enabling students to engage with activities in the context of the book material.
  • Media Integration: Seamlessly play videos and audio within the eReader, complete with transcript support for enhanced accessibility.
  • Jump to Page: Create direct links to specific pages, allowing quick navigation within the eReader.

Integration Specifications

Launch Parameters

  • The eReader URL is structured into four key components:
    baseURL/{Program-Code}/{Product-Class}/{Product-Code}
    • Base URL: https://inlingua-reader.comprodls.com
    • Program Code: The unique identifier for the specific program.
    • Product Class: Specifies whether the eReader is for the Student Edition or Teacher Edition
    • Product Code: The unique identifier for the specific title.
  • To specify a page number to be launched:
    Use the page query parameter in the eReader URL. For example, to launch page 5:
    https://inlingua-reader.comprodls.com/trial_english1step1/student-edition/trial_eng1_step1_se?page=5

Embedding the eReader in Your Application

To embed the eReader wihtin your application, follow these steps:

  • Create an iframe:
    • Set the src attribute of the iframe to the desired eReader URL, e.g.,
      https://inlingua-reader.comprodls.com/trial_english1step1/student-edition/trial_eng1_step1_se
  • Enable Fullscreen:
    • Add the allowfullscreen attribute to the iframe to allow users to switch to fullscreen mode.
  • Embed Mode:
    • Use the embed query parameter in the eReader URL when launching within an iframe.
  • Specify Page Number:
    • Use the page query parameter to start at a specific page.

Sample iframe code:
<iframe src="https://inlingua-reader.comprodls.com/trial_english1step1/student-edition/trial_eng1_step1_se?page=2&embed" allowfullscreen></iframe>
